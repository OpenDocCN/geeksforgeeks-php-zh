# PHP 和 DENO 的区别

> 原文:[https://www . geesforgeks . org/PHP 和-deno 之间的区别/](https://www.geeksforgeeks.org/difference-between-php-and-deno/)

**PHP:** PHP 代表超文本预处理器，是一种脚本语言。PHP 是一种通用的脚本语言。PHP 是拉斯马斯·勒德尔夫在 1994 年创建的。PHP 基本上是在服务器端用于开发 web 应用程序。PHP 有一些框架，比如 laravel、Symfony 等。请注意，PHP 是一种脚本语言，而不是框架。

**DENO:** DENO 是 JavaScript 和 typescript 的简单而安全的运行时，这两者都使 JavaScript 能够在服务器端 JavaScript 上运行，从而使其有足够的能力运行服务器。DENO 可以认为是 Node.js 的升级，DENO 的开发者和 Node.js 的开发者一样，也就是 Ryan Dahl。它在服务器端用于制作快速、可扩展和健壮的应用程序。请注意，它不是一种语言，而是一个运行时环境，用于使用 chromes V8 引擎的 JavaScript 或 typescript，并且内置于 Rust 中。
