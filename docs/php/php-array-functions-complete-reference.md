PHP Arrays 是一种数据结构类型，它允许在一个变量下存储多个相似类型的元素，省去了为每个数据创建不同变量的工作。数组有助于创建相似类型的元素列表，可以使用它们的索引或键来访问这些元素。
允许数组函数以各种方式交互和操作数组元素。PHP 数组内置函数用于简单的多维数组。

**安装:**这些功能在任何安装中都不需要。这些都是 PHP 核心的部分。
