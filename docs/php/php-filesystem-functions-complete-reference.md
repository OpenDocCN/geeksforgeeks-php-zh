Filessystem 函数用于访问和操作文件系统。 这是 PHP 代码的一部分，因此不需要安装这些函数。
访问系统上的文件时，将使用文件路径。 在 Unix 系统上，使用正斜杠(/)作为目录分隔符，在 Windows 平台上，同时使用正斜杠(/)和反斜杠(\)。
